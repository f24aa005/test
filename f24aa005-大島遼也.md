関数を使うと、処理を分割することができる
```
void 関数名(){
    処理
}
```
分割したり、同じ処理を共通化したり

>変数には有効範囲（スコープ）がある。
```
void main(){
    int a = 0;
    print(a);

    syori1();
    syori2();
}

void syori1(){
    print(a);   //エラー
}

void syori2(){
    int a = 10; //mainのaとは別
    print(a);
}
```
関数の中で処理した値を、呼び出したもとに戻すことができる。

>voidは「戻り値がない」という意味

呼び出し元から値を渡すには、引数を使う
引数は好きな数だけ、用意することができる。

処理と値をひとまとめにするプログラミングの機能を`「クラス」`という

>クラスの中では
- 「変数」　→　「フィールド」
- 「関数」　→　「メソッド」
